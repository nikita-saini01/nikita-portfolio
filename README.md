# Nikita Saini — Portfolio

A dark, animated personal portfolio site showcasing my work as an **AI/ML Engineer** & **Full-Stack Developer**.

## ✨ Features

- Interactive 3D particle background (Three.js)
- Rotating role typewriter animation
- 3D-styled headline text with layered depth effect
- Smooth entrance animations (GSAP)
- Fully responsive with mobile hamburger navigation
- Custom font pairing (Space Grotesk, Instrument Serif, Outfit, JetBrains Mono)

## 🛠️ Built With

- HTML5 & CSS3
- Vanilla JavaScript
- [Three.js](https://threejs.org/) — 3D particle field
- [GSAP](https://gsap.com/) — animations

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
open index.html
```
